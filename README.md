### 1. Standard parameters for Ford Focus 2005
*These columns represent the manufacturer's official performance standards for the Ford Focus 2005.*

* **`Moc_KM` (Rated Power):** The engine's maximum power output expressed in Metric Horsepower (KM).
* **`Moc_obr` (Power RPM):** The rotational speed (Revolutions Per Minute) at which the maximum power is achieved.
* **`Mom_Nm` (Rated Torque):** The maximum torque output expressed in Newton-meters (Nm).
* **`Mom_obr` (Torque RPM):** The rotational speed at which the maximum torque is available.
